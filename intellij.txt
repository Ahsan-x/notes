*Shortcuts*
Refactor
    *F6 # Move element (class/field/static method/...)*
    *Shift + F6 # Rename*
    *CMD + F6 # Refactor*
    *Shift + F6 # Rename*
    *CMD + SHIFT + C # move to a constant*
    *CMD + SHIFT + F # move to a field*
    *CMD + SHIFT + P # move to a parameter*
    *CMD + SHIFT + M # move to a method*
    *CMD + SHIFT + V # move to a local variable*


Move
    *ALT + SHIFT + UP/DOWN # move line up down*
    *ALT + SHIFT + UP/DOWN # move line up down*
    *CMD + SHIFT + UP/DOWN # move block up down*
    *CMD + SHIFT + UP/DOWN # move block up down*
    *ALT + SHIFT + J # join lines
    *CMD + backspace* # deletes line
    *CMD + shift + N* # move selected text to a scratch file

    *ALT + F7 # Find usages*
    *CMD + B # Go to declaration*
    *ALT + UP/DOWN # select/deselect within string/method invocation*
    *CMD + F8 # Enable/disable break point*

Menus
    *CMD + E # menu of recently opened files*
    *CMD + N # menu to create constructor, getters, override methods*
    *CMD + shift + T # create new test class*


Git
    *CMD + K # git commit*
    *CMD + T # git pull*
    *CMD + ALT + Z # revert file*


Find
    *CMD + R # replace*
    *CMD + F # find*

Open
    *CMD + O # open class by name*
    *CMD + SHIFT + O # open file by name*
    *CMD + SHIFT + A # find action by name*
    *CMD + NUMBER (3-9) # Go to tab with that number (debug, run, version control, ...)*
    *CMD + J # shows shortcuts*


Errors
    *ALT + ENTER # suggestions about fixing compiling error*
    *CTRL + SPACE # find reference in classpath*


Run test
    *CTRL + R # run last run test*
    *CTRL + SHIFT + R # run selected test / test class / files.*
    *CTRL + D # debug last test*
    *CTRL + SHIFT + D # debug selected test / test class / files.*


Move
    *FN + left/right # go to start line / end line*
    *ALT + left/right # move word*
    *CMD + L # goto line*
    *F2 # go to next error/warning *
    *CMD + alt + up/down # go to next element in list*

MultiCursor
    *CTRL + G # select next that matches current selection*
    *CTRL + SHIFT + G # deselect previous next that matches current selection*
    *CTRL + CMD + G # select all that matches current selection*
    *ALT + DOWN # multi line cursor*
    *CMD + D # duplicate selected text*
    *CMD + backspace # delete line*
    *   CMD + shift + U # change case of selected text*

Autocomplete
    *Tab  # finishes line (adds bracket and semicolon)*
    *Shift + ALT + Enter # finish if (adds bracket, new line and curly bracket)*
    *psf + enter # public static final*
